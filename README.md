## запуск проекта

```bash
# установка зависимостей
npm install

# запуск dev сервера
npm run dev
```

открой [http://localhost:3000](http://localhost:3000) в браузере.

## команды

```bash
npm run dev      # разработка
npm run build    # сборка
npm run start    # production
npm run lint     # проверка кода
```

## технологии

- Next.js 14
- TypeScript
- React Three Fiber
- Tailwind CSS
- Framer Motion
